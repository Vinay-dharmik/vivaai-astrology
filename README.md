# 🌌 VivaAI Astrology — Premium Vedic Astrology & SaaS Platform

VivaAI is a high-performance, conversion-optimized Vedic Astrology SaaS platform. It combines traditional Vedic principles with modern astronomical computations to deliver hyper-accurate birth charts, compatibility reports, and daily panchang.

Built with Next.js 15, optimized for Core Web Vitals, and scaled with programmatic SEO (830+ dynamic landing pages) and hybrid monetization (lazy-loaded Google AdSense + Razorpay freemium paywalls).

🔗 **Live Platform:** [vivaai.in](https://vivaai.in)

## 🌟 Key Features

*   **Programmatic SEO Engine (837+ Pages):** Programmatically pre-rendered static routes for maximum organic reach.
    *   **729 Nakshatra Compatibility Pages** (`/nakshatra-compatibility/*`) — capturing high-intent marriage and relationship queries.
    *   **108 Planet-in-House Pages** (`/astrology/*`) — targeting specific planetary influence searches (e.g., *Sun in 10th House*).

*   **Premium Freemium Paywall:** 
    *   Generates comprehensive base charts for free.
    *   Features a gated, premium detailed report (Manglik analysis, Dasha periods, remedies, and career paths) accessible via a seamless **Razorpay payment checkout (₹49)**.
*   **Cosmic PDF Report Generator:** Client-side, dynamic PDF generation (`jsPDF`) rendering custom dark-gold styled astrology reports instantly after payment verification.
*   **Viral Acquisition Loops:** One-click WhatsApp share links with pre-filled, personalized Kundali statistics (Lagna, Nakshatra, Mahadasha) to trigger peer-to-peer growth.
*   **Technical SEO & Rich Snippets:** Fully integrated JSON-LD schemas (`SoftwareApplication`, `FAQPage`, `Article`, and `BreadcrumbList`) to claim star ratings and rich results on Google SERP.
